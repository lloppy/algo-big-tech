## Новое задание без решения

### Условие задачи

Дана строка s, состоящая из строчных латинских букв. Нужно найти длину самой длинной подстроки, которая не содержит повторяющихся символов.

### Пример

Вход: s = "abcabcbb"
Выход: 3

Вход: s = "bbbbb"
Выход: 1

Вход: s = "pwwkew"
Выход: 3

### Ограничения

* 0 <= s.length <= 5 * 10^4
* s состоит из строчных латинских букв.
