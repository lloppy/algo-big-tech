# 1. Решение

```kotlin
var a = p
var b = q

while (a != b) {
    a = if (a == null) q else a.parent  // A идет: p → parent → ... → null → q → ...
    b = if (b == null) p else b.parent  // B идет: q → parent → ... → null → p → ...
}
```
